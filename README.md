# personal-site

Personal portfolio site for Liam Davis.

**Live:** [lmdav.com](https://lmdav.com)

## Tech Stack

- TypeScript
- Next.js 16 (App Router)
- React 19
- Tailwind CSS v4
- Lenis (smooth scroll)

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view.
